# Calculateur d'UFC

A lightweight web-based calculator for UFC (Unités Formant Colonies) used in microbiological analyses.

## Features

- Input by colony type (A–E)
- Supports:
  - Direct concentration
  - Filtration (10 ml and 100 ml)
- Handles "envahie flore" conditions
- Calculates:
  - Colonies per type
  - Total colonies
  - Colonies requiring confirmation

## Installation

No installation required.
Open index.html in a modern web browser.

## Technologies 

- HTML5
- JavaScript (Vanilla)
- Bootstrap 4

## File Structure

- index.html – UI and layout
- UFC_V2.2.js – calculation logic
- style.css – optional custom styling

## Intended Use

Laboratory or training use for microbiological colony counting.
Not intended for clinical decision-making without validation.
